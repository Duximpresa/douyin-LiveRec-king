# Changelog

## 0.3.0 - 2026-06-23

- Added Bilibili live-room monitoring and platform-owned request headers/Cookies.
- Added configurable retry limits, backoff intervals, and global retry concurrency.
- Added recording history recovery, ffprobe inspection, safe TS-to-MP4 remuxing, and temporary-file review.
- Added recording statistics and UTF-8 BOM CSV exports.
- Split application services and UI pages while preserving compatibility imports.
- Added deterministic concurrency tests, local FFmpeg integration checks, and Windows release validation.

## 0.2.1

- Added the first reliable recording core, typed recording events, graceful shutdown, recording history, and network retry handling.
